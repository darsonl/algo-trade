Create `screener/macro.py` with these exact functions:

1. `format_spy_trend(one_month_return: float) -> str`:
   - If `one_month_return >= 0`: return `f"Bullish (+{one_month_return * 100:.1f}%)"`
   - If `one_month_return < 0`: return `f"Bearish ({one_month_return * 100:.1f}%)"`
   (The negative sign is already in the float, so format naturally produces "Bearish (-1.4%)")

2. `format_vix_level(vix_close: float) -> str`:
   - If `vix_close < 20`: label = "Low volatility"
   - If `20 <= vix_close <= 30`: label = "Elevated volatility"
   - If `vix_close > 30`: label = "High volatility"
   - Return `f"{vix_close:.1f} ({label})"`

3. `compute_52w_position(current_price: float | None, low_52w: float | None, high_52w: float | None) -> str`:
   - If any of current_price, low_52w, high_52w is None, return "N/A"
   - If high_52w == low_52w, return "N/A" (avoid division by zero)
   - `pct = (current_price - low_52w) / (high_52w - low_52w)`
   - `pct_int = round(pct * 100)`
   - If pct >= 0.80: return `f"Near high ({pct_int}%)"`
   - If pct <= 0.20: return `f"Near low ({pct_int}%)"`
   - Else: return `f"Mid-range ({pct_int}%)"`

4. `fetch_macro_context() -> dict`:
   - Uses `yf.Ticker("SPY")` and `yf.Ticker("^VIX")`
   - SPY: fetch `history(period="1mo")`, compute return as `(last_close - first_close) / first_close`
   - VIX: fetch `fast_info["lastPrice"]` or fall back to `history(period="5d")` last close
   - Wrap entire body in try/except. On ANY exception: `logger.warning("Failed to fetch macro context: %s", exc)` and return `{"spy_trend": None, "vix_level": None}`
   - On success: return `{"spy_trend": format_spy_trend(spy_return), "vix_level": format_vix_level(vix_close)}`
   - Do NOT use tenacity retry on this function (per D-06: failure is acceptable, just return None)

Module-level: `import logging`, `import yfinance as yf`, `logger = logging.getLogger(__name__)`
