**database/queries.py** -- Add 6 new functions at the end of the file, following the existing `get_connection` open/commit/close pattern:

1. `create_position(db_path: str, ticker: str, shares: float, avg_cost_usd: float) -> int`
   Uses `INSERT INTO positions (ticker, shares, avg_cost_usd, entry_date, status) VALUES (?, ?, ?, date('now'), 'open') ON CONFLICT(ticker) DO UPDATE SET shares=excluded.shares, avg_cost_usd=excluded.avg_cost_usd, entry_date=date('now'), status='open', last_price=NULL, last_updated=NULL`
   Returns `cursor.lastrowid`.

2. `update_position(db_path: str, ticker: str, new_shares: float, buy_price: float) -> None`
   SQL: `UPDATE positions SET shares = shares + ?, avg_cost_usd = (shares * avg_cost_usd + ? * ?) / (shares + ?) WHERE ticker = ? AND status = 'open'`
   Parameters: `(new_shares, new_shares, buy_price, new_shares, ticker)`

3. `get_open_positions(db_path: str) -> list[sqlite3.Row]`
   SQL: `SELECT * FROM positions WHERE status = 'open' ORDER BY entry_date ASC`

4. `has_open_position(db_path: str, ticker: str) -> bool`
   SQL: `SELECT id FROM positions WHERE ticker = ? AND status = 'open'`
   Returns `row is not None`.

5. `close_position(db_path: str, ticker: str) -> None`
   SQL: `UPDATE positions SET status = 'closed' WHERE ticker = ? AND status = 'open'`

6. `upsert_position(db_path: str, ticker: str, shares: float, price: float) -> None`
   Calls `has_open_position`; if True -> `update_position(db_path, ticker, shares, price)`, else -> `create_position(db_path, ticker, shares, price)`.

All functions must have docstrings (per Phase 3 convention DOC-05). Import `sqlite3` is already at the top of queries.py.
